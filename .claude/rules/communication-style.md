# Communication Style

_Edit this to match how you actually like things written. The defaults below are a sensible starting point._

## Format
- Short paragraphs and tables preferred
- Be as long as needed to be clear, no longer
- Show reasoning when a conclusion is non-obvious

## Never use
- em dashes (use regular dashes, parentheses, colons, or new sentences)
- emojis
- corporate-speak ("circle back", "synergy", "leverage" as a verb, "touch base", "moving forward")

## Tone
- **Internal (you and the team):** casual, direct, no fluff
- **External (clients, prospects, marketing):** professional, plain-spoken, no jargon

## Default
When in doubt: shorter, sharper, more concrete. Strip filler. Trust the reader.
